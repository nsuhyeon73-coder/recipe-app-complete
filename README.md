# Recipe Cook - 맛있는 요리 레시피 🍳

전 세계의 맛있는 레시피를 한 곳에서! React와 Vite로 만든 레시피 검색 웹 애플리케이션입니다.

## 🚀 설치 방법

```bash
# 1. 의존성 설치
npm install

# 2. 개발 서버 실행
npm run dev

# 3. 브라우저에서 http://localhost:5173 접속
```

## 📦 빌드

```bash
npm run build
```

## ✨ 주요 기능

- 🎲 랜덤 레시피 추천
- 🔍 레시피 검색
- 📂 카테고리별 분류
- 🌍 국가별 요리
- 🌐 한글/영어 언어 전환
- 📱 반응형 디자인

## 🛠 기술 스택

- React 18
- Vite
- Tailwind CSS
- TheMealDB API
